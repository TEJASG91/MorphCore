# Progress log

A short visual history of the engine, one entry per milestone. This is portfolio and blog
fuel — by M4 you have a complete "watch it grow" reel.

## Convention

For each milestone, drop files here named `NNN-mN`:

```
docs/progress/
├── 001-m1.mp4     (or .gif)   engine core — random cloud assembling into a face
├── 001-m1.png                 a representative still
├── 002-m2.mp4                 living face — breathing, drifting, tracking the cursor
├── 002-m2.png
└── 003-m3.mp4                 scroll morph — face → name → skills
```

## How to capture

- **Screen recording:** any recorder works; on Windows, Xbox Game Bar (`Win+G`). Record the
  browser with the dev HUD off. Trim to ~5–10 s of the key motion.
- **GIF:** convert the clip (e.g. ffmpeg or an online converter). Keep it small — a few MB.
- **Still:** a clean frame that reads well as a thumbnail.

Keep clips short and loop-friendly; they are teasers, not walkthroughs. The M1 and M2 states
still exist in the tagged history if you want to record them retroactively.
